# Godot 2D Fundamentals Workshop

This repository is for the Godot 2D Fundamentals Workshop, where students learn the basics of Godot by building a simple 2D game together.

## About the Workshop

In this session, students will learn:

- Navigating the Godot Editor  
- Understanding nodes and scenes  
- Working with sprites and collisions  
- Basic physics  
- GDScript fundamentals  
- Implementing player movement  
- Creating simple gameplay interactions  

By the end of the workshop, each student will have built the same base 2D project.
